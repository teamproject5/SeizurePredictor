About Us

The Seizure Predictor is an analysis software created to help patients know when a seizure is about to occur. The software product has been tested using Kaggle real-world data taken from humans and dogs.

This software does not stop a seizure from occurring, but instead warns the patient that one is oncoming. This gives the patient adequate time to decrease the chances of having an injury or fatal event happen during a seizure.
